## NHN Cloud > Public API > 개요

### Public API는?
- Public API는 NHN Cloud에서 제공하는 API로, NHN Cloud의 프레임워크 서비스를 외부에서 사용할 수 있도록 제공합니다.
- Public API를 사용하려면, API 인증을 통해 발급받은 토큰을 사용하여 API를 호출해야 합니다.
