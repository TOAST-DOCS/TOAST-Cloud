# サービスAPI

**NHN Cloud > Public API > サービスAPI**

서비스 API는 NHN Cloud에서 제공하는 개별 서비스와 해당 서비스의 리소스를 외부 시스템 또는 사용자 애플리케이션에서 제어하거나 연동할 수 있도록 지원하는 API입니다.
